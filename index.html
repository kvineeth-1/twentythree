Sticky
  1. Position relative is related to browser screen or root parent container
  2. Position absoulute is related to parent if it doesn't find parent than it will go for root
  3. Position fixed is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
  4. Position sticky is combination of position relative and fixed
  5. Position static defines the position of a given box essentially as an unpositioned element
